The VC Working Group invites implementations of the following Candidate Recommendation Snapshot:

  • Securing Verifiable Credentials using JOSE and COSE. This specification defines how to secure credentials and presentations conforming to the Verifiable Credential data model with JSON Object Signing and Encryption (JOSE), Selective Disclosure for JWTs, and CBOR Object Signing and Encryption (COSE). This enables the Verifiable Credential data model to be implemented with standards for signing and encryption that are widely adopted.

Comments are welcome via the GitHub issues by 25 May 2024.
